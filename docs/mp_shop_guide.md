# 🎫 MPチケット & MPショップ ガイド

## MPチケットとは？
**VCで過ごすともらえるポイント**です。集めるとMPショップで色々な特典と交換できます。

- **VCランクが1レベル上がるごとに1枚**もらえます
- VCランクは **VCに合計5時間いるごとに1レベル**上がります（＝5時間で1枚のイメージ）
- ※一部のカテゴリのVCは、時間のカウントが **1/3** になります

## 🎫 チケットの確認方法
MPショップパネルの **「🎫 チケットを確認」ボタン** を押すと、あなたの所持枚数が **あなただけに見える形** で表示されます。

## 🎁 交換できる商品（MPショップ）
パネルの **セレクトメニュー** から商品を選ぶと交換できます。

### 🔄 お試し個通のリセット … **20枚**
お試し個通で **一度誘った相手に、もう一度誘えるようになります**（誘った履歴をリセット）。

### 📝 個人専用テキストチャット作成 … **10枚**
自分専用のテキストチャンネルを作れます。
- 作成時に **チャンネル名** と **閲覧できるロール（男性 / 女性）** を指定します（1つ以上必須）
- **1人につき1つまで**（作ったチャンネルを削除すれば、また作れます）

### 🎨 ロール作成 … **5枚**
オリジナルのロールを作って自分に付けられます。
- 作成時に **ロール名** と **色（`#RRGGBB` 形式・例：`#FF0000`）** を指定します
- 色は空欄でもOK（色なしになります）

### 📷 雰囲気写真の閲覧権 … **10枚**
「雰囲気写真」チャンネルを閲覧できるようになります。
- ⚠️ **24時間以内に「雰囲気写真」チャンネルへ画像を投稿しないと、閲覧権は没収されます**
- 没収された場合は、もう一度交換すれば再取得できます

### 😀 サーバー絵文字を追加 … **5枚**
好きな画像をサーバーのカスタム絵文字にできます。
- 作成時に **絵文字の名前**（英数字と `_`・2〜32文字）と **画像** を指定します
- 画像は **256KB以下**（png / jpg / gif）

## 交換内容の早見表
- 🔄 **お試し個通のリセット** … 20枚（誘い履歴をリセット）
- 📝 **個人専用テキストチャット作成** … 10枚（名前・閲覧ロールを指定／1人1つ）
- 🎨 **ロール作成** … 5枚（名前と色を指定して作成・付与）
- 📷 **雰囲気写真の閲覧権** … 10枚（24時間以内に画像投稿しないと没収）
- 😀 **サーバー絵文字を追加** … 5枚（画像をアップロードして絵文字化）

## ❓ よくある質問・注意点
- **チケットが足りない場合は交換できません**（チケットは消費されません）
- 交換の結果メッセージは **あなたにだけ** 表示されます
- 交換に失敗した場合（絵文字枠が満杯・画像が大きすぎる等）、**チケットは消費されません**
- 同じ商品を続けて交換したいときは、もう一度選び直せばOKです
