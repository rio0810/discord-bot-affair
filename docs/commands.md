# 🤖 Mimoza コマンド一覧

## 👤 一般ユーザー向け
- `/topic` … 雑談のネタをランダムに出します
- `/rank` … 現在のVCランクと滞在時間（ランクカード画像）を表示します

## 🛠️ 管理者専用
> 「運営ロール」を持つ人だけが使用可。

**パネル設置**
- `/set_appeal_panel` … 面接・面談パネルを設置（男性：面接録音／女性：プロフィール記載）
- `/set_call_panel` … 個通募集パネルを設置
- `/set_role_panel` … 雑談 / 恋愛ロールの切替パネルを設置
- `/set_mp_panel` … MPチケット交換パネルを設置

**MPチケット管理**
- `/mp_give` … 指定ユーザーにMPチケットを配布
- `/mp_take` … 指定ユーザーからMPチケットを没収
- `/mp_list` … メンバーのMPチケット所持数を表示

**運営・管理ツール**
- `/admin_block_list` … 個通のブロック状況を確認
- `/newcomer_list` … 新人ロールを持つメンバーを一覧表示

## ✨ コマンド以外の主な自動機能
- **VC時間の計測 / MPチケット配布** … VC滞在でランクが上がり、1レベルにつきMPチケット1枚を獲得
- **テキストレベル** … メッセージ投稿で加算（`/rank` のカードに表示）
- **プロフィール作成** … 面接・面談パネルのボタンから作成（男性は面接チャンネルへの録音投稿で審査へ）
- **個通マッチング** … 個通募集パネルのボタンから相手を誘う（ブロック・人数制限・お試し通話など）
- **一時VC（フリールーム）** … トリガーVCに入ると個人用VCが自動作成され、名前変更パネルが出る
- **MPショップの交換** … MPチケット交換パネルのボタン/セレクトから商品と交換
